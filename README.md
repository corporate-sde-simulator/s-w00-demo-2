## How It Works

Each demo is a self-contained mini-project with:
- **TICKET.md** — Your JIRA ticket (just like a real company)
- **src/** — The code you need to fix
- **docs/** — Architecture docs and a learning guide
- **tests/** — Tests to verify your fix
- **.context/** — Meeting notes and PR review comments
